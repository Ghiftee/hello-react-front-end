## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- Yarn

### Setup

```bash
git clone https://github.com/Ghiftee/hello-react-front-end.git
cd ./hello-react-front-end
```

### Install

```bash
npm install
```

### Usage

```bash
npm start
```
